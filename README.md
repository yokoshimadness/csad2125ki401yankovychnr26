# "Computer systems automated design" works
This repository contains works for the course "Computer systems automated design".

## Task 
Develop a simple game;

## Menu
The game should have a menu that allows changing parameters;

### Play Modes:
- Man vs AI
- Man vs Man
- AI vs AI:
  - Random move;
  - Win strategy;

### Actions:
- New;
- Save;
- Load;
